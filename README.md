# motion_detection
detecting the motion and gives indication and beep sound
Motion Detection using OpenCV

Overview
Detects motion in real-time using webcam.
Highlights movement with green boxes and plays a beep alert

Technologies Used
Python 3
OpenCV (cv2)
Platform module
Threading
Winsound / ALSA (for beep sound)

Installation
pip install opencv-python
python test.py

Future Enhancements
Add email/SMS alerts
Save detected motion video/images
